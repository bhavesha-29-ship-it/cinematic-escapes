## 🚀 Project Features
This project was built to fulfill all mandatory requirements of the Assignment 1 Rubric:
**Minimum 5 Pages:** Includes Home, About, Packages (Services), Gallery, and Contact pages.
**Responsive Design:** Fully fluid layout that adapts to mobile, tablet, and desktop screens using CSS media queries and a custom grid system.
**Basic SEO Implementation:** Includes semantic HTML, unique `<title>` tags, meta descriptions, meta keywords, proper heading hierarchy (`<h1>`, `<h2>`, `<h3>`), and descriptive `alt` text for all images.
**JavaScript Interactivity:** Features a mobile hamburger navigation menu toggle and client-side form submission handling on the Contact page.
**Map/Location Section:** Includes an embedded, stylized interactive Google Map highlighting the agency's headquarters.
**Cinematic Aesthetic:** Custom UI/UX focusing on deep reds, golden hour tones, and high-contrast styling.
